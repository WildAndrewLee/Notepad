About Notepad

Notepad is a lightweight copy-paste/code snippet/whatever you want host built using Node.js for its native async support and LevelDB for it's simple yet performant key-value storage.

Everything you do on Notepad is completely anonymous. We don't keep any logs (except to rate-limit you of course). The current rate-limit on code sharing is 15 POST request every 60 seconds.

Want to help contribute? Notepad is open source! Feel free to fork it over at Github.

Notepad API Documentation

Notepad utilizes a RESTful API. The API's routes are as follows:

Share Code

This endpoint allows you to share things on Notepad and returns a JSON object on success. The snowflake attribute is an identifier used to lookup whatever you're sharing. 

POST /share HTTP/1.1
Content-Type: multipart/form-data
Response-Type: application/json
Payload: code=[string]

Example Call

curl -F "code=$(cat snippet.js)" https://notepad.somoe.moe/share

Example Response

{
    "snowflake": [string]
}
