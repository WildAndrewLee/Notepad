Share Something About Notepad RESTful API

API Documentation

Notepad utilizes a RESTful API. The API's routes are as follows:

Share Code

This endpoint allows you to share things on Notepad and returns a JSON object on success. The snowflake attribute is an identifier used to lookup whatever you're sharing. Snowflakes are practically unique. 

POST /share HTTP/1.1
Content-Type: multipart/form-data
Response-Type: application/json
Payload: code=[string]

Example Call

curl -F "code=$(cat snippet.js)" https://notepad.somoe.moe/share

Example Response

{
    "snowflake": [string]
}
